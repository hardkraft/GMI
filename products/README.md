## Description

[Products](https://github.com/hardkraft/GMI/tree/main/products) the test project repository.

Simple CRUD App with basic localization.
Change the language in sever/config.ts

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
